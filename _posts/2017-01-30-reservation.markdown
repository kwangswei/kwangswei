### 연박 할인
- 연박 시 침구를 그대로 사용하실 경우 세탁비만큼 할인해드립니다!
  - ex) 2박 시 둘째 날에 침구를 그대로 사용할 경우 - 5천원 할인 = 7만원 x 2일 - 할인 5천원 = 13.5만원
  - ex) 3박 시 둘째 날, 셋째 날에 침구를 그대로 사용할 경우 - 1.5만원 할인 = 7만원 x 3일 - 1.5만 = 19.5만원
- 기본적으로 매일 베게커버, 매트리스시트, 침구커버를 세탁합니다.
